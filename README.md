# UNO

## Compilation

### Requirements

- [xmake]\: build system.

[xmake]: https://github.com/xmake-io/xmake/

### Instructions

```ps1
# Build
xmake -y

# Run (requires build first)
xmake run -w .

# Test
xmake test -w .

# Generate compilation database
xmake project -k compile_commands

# Generate documentation
doxygen
```
